# OurCompany

## 소개
영진 전문대 2017년 CP-B반에서 캡스톤 디자인을 하기위해

학생들을 모여 만든 OurCompany입니다.

우리는 캡스톤 디자인을 하면서 프로젝트를 계획과 설계, 개발을 함으로써

회사 업무에 대해 선행학습을 진행합니다.

우리의 프로젝트는 My Love Pet으로 안두이노를 이용하여 동물에게서 정보를 얻고

그 정보를 앱과 웹에서 이용하여 반려 동물을 쉽게 관리하고 육양 할 수 있도록 합니다.

## <a name = "strategy">프로젝트 추진 전략</a>

 1. 1학기가 끝나기 전에 모든 프로젝트 완료

 2. 매주 월요일 6시 전조원이 미팅을 통해 주간계획 수립

 3. 리눅스 기반의 APM(Apache+PHP+Mysql)+ Tomcat연동을 통한 실무 능력 향상

 4. 완벽한 객체지향으로 확장성을 위해 개발 진행

 5. 실제 사용가능한 어플리케이션 개발
